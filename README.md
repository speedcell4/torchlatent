## Requirements

- Python 3.8
- PyTorch 2.0

## Installation

`python3 -m pip torchlatent`

## Latent Structures

- [x] Conditional Random Fields (CRF)
- [x] Tree CRF
- [ ] Non-Projective Dependency Tree (Matrix-tree Theorem)
- [ ] Probabilistic Context-free Grammars (PCFG)
- [ ] Dependency Model with Valence (DMV)
